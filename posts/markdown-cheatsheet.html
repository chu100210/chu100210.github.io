Markdown 写作指北

2026-07-10  ·  工具

Markdown 是我最喜欢的写作格式——语法简单，纯文本，任何编辑器都能打开，还能轻松转成 HTML。这篇文章整理一下常用语法和几个提升效率的工具。

编辑器推荐

我主要用 VS Code 写 Markdown，搭配这几个插件：

  • Markdown Preview Enhanced — 实时预览，支持数学公式、流程图、PPT 导出
  • markdownlint — 语法规范检查，改掉坏习惯
  • Paste Image — 截图直接粘贴，自动保存到图床目录

几个小技巧

  1. 标题层级不要跳——一级标题应该是文章标题，正文从二级开始
  2. 代码块写清楚语言——方便自己和读者阅读，也方便某些工具高亮
  3. 段落间空一行——很多解析器依赖空行来分段落
  4. 链接用引用式——长文里把链接放到文末，正文更干净

用好 Markdown，写作效率能提升一大截。毕竟少操心排版，多专注于内容本身才是正事。
