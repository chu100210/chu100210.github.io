Hello World — 第一篇博客

2026-07-18  ·  随笔

折腾了好久，终于把这个博客搭起来了。这是我的第一篇博文，算是「Hello World」。

为什么要建博客

其实想了很久要不要写博客。一方面觉得现在短视频、公众号这么多，还有没有人看文字博客？另一方面又觉得，写东西最大的受益者其实是写作者自己。

写下来的过程，就是一个把模糊的想法变清晰的过程。很多东西你以为自己懂了，真要写清楚就会发现到处都是漏洞。所以写博客首先是为了整理自己的思路。

其次，写了这么多年代码，踩了很多坑，有些经验不记下来就忘了。就算只帮到一个人，也值了。

这个博客会写什么

暂时想到几个方向：

  • 技术教程 — 前端、后端、DevOps，实操性的内容
  • 工具推荐 — 好用的软件、插件、效率工具
  • 读书笔记 — 技术书和非技术书都有
  • 碎碎念 — 偶尔记录下生活和想法

技术选型

这个博客用的是最朴素的方案：

  • GitHub Pages — 免费托管，自动部署
  • 纯 HTML + CSS — 没有构建工具，没有框架，推上去就能看
  • 手写样式 — 自适应亮色/暗色模式，移动端友好

没选 Jekyll 或者 Hexo 这些静态站点生成器，是因为不想折腾 Ruby 和 Node 环境。纯手写 HTML 虽然原始，但胜在简单可控，写文章就是新建一个 HTML 文件的事。

大道至简。能用 HTML 解决的问题，就不要引入 147 个 npm 依赖。

以后如果文章多了手动维护太累，再考虑引入一个简单的构建工具。

第一篇就到这里。感谢你看到这行字，希望这个博客能坚持更新下去 🚀
