国际权威

VirusTotal

Google 旗下 · 全球最大在线病毒检测平台

VirusTotal 是网络安全行业的事实标准。它聚合了卡巴斯基、赛门铁克、McAfee 等超过 60 个杀毒引擎同时扫描，被全球安全研究者和企业安全团队作为首选判定工具。

  • 检测引擎数：60+
  • 所属：Google (Alphabet Inc.)
  • 判定：全部引擎未检测到恶意内容

国际权威

Mozilla Observatory

Mozilla 基金会 · Firefox 浏览器开发商

Mozilla 是全球互联网标准的重要推动者，其 Observatory 工具被全球开发者视为网站安全配置的黄金标准。它检测 HTTP 安全头、CSP 策略、TLS 配置等多个维度的安全实践。

  • 检测维度：CSP / HSTS / TLS / XSS防护 / 点击劫持防护
  • 所属：Mozilla Foundation（非营利组织）
  • 判定：安全配置评分达标

国际权威

HSTS Preload

Google 维护 · 浏览器强制加密列表

HSTS Preload 列表由 Google 维护，内置于 Chrome、Edge、Firefox 等所有主流浏览器核心代码中。进入该列表后，浏览器将强制使用 HTTPS 访问，从底层杜绝中间人攻击和 SSL 剥离攻击。

  • 维护方：Google Chrome Team
  • 覆盖范围：Chrome / Edge / Firefox / Safari 全平台
  • 判定：HSTS 配置合规

国内权威

腾讯安全检测

腾讯 · 国内互联网安全领军

腾讯安全团队是国内最大的互联网安全力量之一，为微信、QQ 等数十亿用户提供安全防护。其站点检测引擎基于腾讯自研的恶意 URL 识别模型，对国内网络威胁有最精准的识别能力。

  • 安全团队：腾讯安全（Tencent Security）
  • 覆盖场景：恶意链接 / 钓鱼检测 / 病毒木马
  • 判定：安全，无可疑内容

百度安全检测

百度 · 中国最大搜索引擎

百度安全检测依托百度搜索引擎的海量数据积累，每天扫描数十亿网页。作为中文互联网最大的流量入口，百度对网站安全性的判断标准被国内用户广泛信赖。
